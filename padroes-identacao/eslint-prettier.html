O que é ESLint e Prettier?

ESLint é uma ferramenta para identificar problemas no código JavaScript. Prettier é um formatador de código.

Configuração Básica:


  // .eslintrc.js
  module.exports = {
    extends: ['eslint:recommended', 'plugin:@salesforce/eslint-config-lwc/recommended'],
    rules: {
      'no-console': 'off',
    },
  };
  
  // .prettierrc
  {
    "singleQuote": true,
    "trailingComma": "es5"
  }
